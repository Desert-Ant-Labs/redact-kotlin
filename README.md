# redact-kotlin has moved

This package has moved to the unified Redact repository:

https://github.com/Desert-Ant-Labs/redact

Use the Maven Central package from the new repo:

```kotlin
implementation("ai.desertant:redact:0.3.0")
```

To bundle the model for offline Android apps:

```kotlin
implementation("ai.desertant:redact-onnx-resources:0.3.0")
```

This repository is archived and no longer maintained.
